# AI成长记录与求职作品集网站

这是一个基于 Next.js 14+ (App Router) 构建的单页个人网站，旨在记录博主在 AI 领域的成长历程，并作为求职作品集展示。

## 项目定位
- **核心属性**：AI成长记录 + 个人求职作品集
- **博主定位**：专注 AI 成长的内向创作者
- **视觉风格**：极简黑白灰，科技感与温度并存

## 技术栈
- **框架**: Next.js 14+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS (v4)
- **动画**: Framer Motion
- **图标**: Lucide React

## 目录结构
- `src/app`: 页面路由
- `src/components`: 页面组件
  - `Hero.tsx`: 封面首屏 (点击解锁)
  - `Navigation.tsx`: 悬浮侧边导航
  - `About.tsx`: 关于我 (包含彩蛋)
  - `Projects.tsx`: 我的作品 (本网站介绍)
  - `Blog.tsx`: 成长日志
  - `Footer.tsx`: 页脚与行动号召
  - `ParticleBackground.tsx`: 全局粒子背景
  - `EasterEgg.tsx`: 隐藏彩蛋

## 功能特性
1.  **单页滚动**: 平滑滚动体验。
2.  **交互动效**: 鼠标跟随粒子、卡片悬浮、点击反馈。
3.  **隐藏彩蛋**:
    - "关于我" -> 点击 "做饭/研究美食" 触发动画。
    - "成长日志" -> 点击标题触发光效。
    - 页面角落隐藏图标 -> 点击弹出暖心文案。
    - 页脚 CTA 按钮 -> 点击触发专属彩蛋。
4.  **响应式设计**: 适配桌面、平板和移动端。

## 部署
本项目可直接部署至 Vercel 或 Netlify。
